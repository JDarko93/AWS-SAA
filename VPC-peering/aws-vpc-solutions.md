## Virtual Private Network (VPN)
- VPN provides an encrypted point to point network device from one device to a network
- VPC appliance can be either hardware or software
- Virtual Network Gateway is a software VPN appliance
- They require at least 1 public IP address
- You can use Pre Shared Keys (PSK) that require both ends of the connection to know the same key 
- Or you can use PKI certificate certification

### AWS Site-to-site VPN
- This is linking on prem networks to AWS cloud
- IPSec is a network security protocol which is used to secure network connections including VPNs
