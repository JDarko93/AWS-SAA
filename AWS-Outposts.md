## AWS Outposts
- AWS services that are running on-prem or edge location
-It allows for local data processing quicker on prem v via the cloud
- Data residency restrictions due to regulatory compliance or contractural implications
- Low latency service access 

### AWS Outposts Servers
- These are rack mountable servers in 1U and 2U options (1U = 1.75 inches in vertical height)
- Supports many AWS services like EC2, ECS, AWS IoT etc

### AWS Outpost racks
- These are entire equipment racks, 80 inches in height
- Include Servers, network switches, network patch panel, power
- Support many AWS services such as EC2, ECS, S3, EBS, RDS, EKS etc
